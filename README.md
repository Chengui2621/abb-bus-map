# ABB班车线路查询系统

厦门ABB班车线路可视化查询系统，基于高德地图API开发。

## 在线访问

🔗 **https://你的用户名.github.io/abb-bus-map/**

> 请将 `你的用户名` 替换为您的GitHub用户名

## 功能特点

- 🗺️ **地图可视化**：基于高德地图，清晰展示所有班车线路
- 🎨 **线路高亮**：多层渲染效果，线路更加醒目
- 📍 **站点标注**：圆点标记 + 序号标签 + 站点名称
- 🔍 **智能搜索**：支持按线路名称、站点名称搜索
- 📊 **线路筛选**：A线/B线快速筛选
- 🕐 **时刻查询**：显示各站点到达时间
- 📱 **响应式设计**：支持电脑和手机访问

## 线路类型

| 类型 | 颜色 | 说明 |
|------|------|------|
| A线 | 🟠 橙色 | 上班班车线路 |
| B线 | 🔵 蓝色 | 上班班车线路 |

## 使用说明

1. **查看线路**：点击左侧线路卡片，在地图上查看完整路线
2. **定位站点**：点击站点列表中的站点，地图自动定位并显示详情
3. **搜索线路**：在搜索框输入线路名或站点名进行搜索
4. **筛选类型**：点击顶部的"A线"/"B线"按钮筛选线路

## 数据来源

- 班车时刻表：日常班车运行时刻表-20260316.xlsx
- 地图服务：高德地图API

## 技术栈

- HTML5 + CSS3 + JavaScript
- 高德地图 JavaScript API 2.0
- GitHub Pages 托管

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/你的用户名/abb-bus-map.git

# 进入目录
cd abb-bus-map

# 启动本地服务器
python -m http.server 8000

# 浏览器访问 http://localhost:8000
```

或者直接双击 `index.html` 在浏览器中打开。
